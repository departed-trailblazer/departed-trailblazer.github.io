发表于 2018-01-04 | 分类于 技术文章

Linux常见目录及作用

bin

保存所有用户都可以使用的系统命令

sbin

保存只有root身份的用户可以使用的系统命令

boot

保存用户的启动数据,应单独分区，保证系统能够有足够的启动空间

dev

保存硬件文件

etc

保存系统的配置文件

home

普通用户的家目录

root

超级用户的家目录

lib

函数库保存目录

media

空目录 外接存储设备的盘符 老式linux系统中 只有mnt外接盘符目录

misc

空目录 外接存储设备的盘符 老式linux系统中 只有mnt外接盘符目录

mnt

空目录 外接存储设备的盘符

proc

内存过载点，不能直接操作

sys

内存过载点，不能直接操作

tmp

临时数据目录

usr

系统资源保存目录
该目录下的bin子目录下保存普通用户可以使用的命令
该目录下的sbin子目录下保存只有超级管理员能够使用的命令

var

保存系统相关文档内容
